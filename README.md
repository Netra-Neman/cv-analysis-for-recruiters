# CV Analysis for Job Recruiter

## Project Overview
CV Analysis for Job Recruiter is a data-driven application designed to assist recruiters in analyzing resumes efficiently. The project focuses on extracting key information from CVs and supporting candidate shortlisting using data analysis and basic ML techniques.

This project aims to reduce manual resume screening effort and improve recruitment efficiency.

---

## Objectives
- Automate resume analysis and information extraction
- Identify relevant skills, experience, and keywords from CVs
- Support recruiters in faster and more accurate candidate shortlisting
- Improve hiring decisions using data-driven insights

---

## Tools & Technologies
- Python
- Pandas, NumPy
- Machine Learning (basic classification / similarity logic)
- Git & GitHub

## How It Works
1. Resume files are provided as input
2. Text is extracted and cleaned
3. Techniques identify skills, experience, and keywords
4. Candidates are evaluated against predefined job requirements
5. Output assists recruiters in shortlisting suitable candidates

---

## Key Features
- Resume text extraction and preprocessing
- Skill and keyword identification
- Candidate relevance scoring
- Simple and scalable logic for screening

---

## Insights Generated
- Identification of most relevant skills per candidate
- Comparison of resumes against job requirements
- Reduction in manual resume screening effort
